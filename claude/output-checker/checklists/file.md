# FILE checklist (docx, pptx, xlsx, pdf)

Labels: 🔴 MISSING · 🟡 FORMAT · 🔵 POLISH (severity scale in SKILL.md).

1. **Structure** — table of contents matches content? Slides in logical order? Sheet names clear?
2. **Formatting consistency** — fonts, sizes, colors, spacing uniform throughout?
3. **Content completeness** — leftover placeholder text? Missing sections? Empty slides/cells?
4. **Data accuracy** — formulas correct? Charts match underlying data? Numbers consistent across sections?
5. **Readability** — text overflow? Badly cropped images? Sensible margins?
