# MATH checklist

Labels: 🔴 ERROR · 🟡 ASSUMPTION · 🔵 RIGOR (severity scale in SKILL.md).

1. **Step validity** — is each step logically justified? Flag any leaps.
2. **Arithmetic** — recompute key calculations independently; don't eyeball them.
3. **Assumptions** — are all assumptions stated? Are they reasonable?
4. **Edge cases** — does the result hold at boundaries (0, ∞, negative)?
5. **Units & dimensions** — consistent throughout?
6. **Final answer** — does it make intuitive sense? Sanity-check with rough estimation.

If the math is correct, confirm: "✅ Derivation verified — all steps valid."
